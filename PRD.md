# BusCatcher — Product Requirements Document

## 1. Overview

**Название:** BusCatcher  
**Тип:** Веб-приложение (PWA)  
**Описание:** Приложение показывает автобусы между выбранными остановками и помогает понять, успеешь ли ты на них, учитывая время на дорогу до остановки.  
**Целевая аудитория:** Пользователи, которые хотят понять, успеют ли на автобус если выйдут сейчас.

---

## 2. User Flow (Новый UX)

```
1. Клик на остановку → Остановка посадки (🚌)
2. Клик на остановку → Остановка высадки (🏁)
3. Клик на карту → Откуда выходишь (🏠)
4. → Показать: автобусы A→B + успеешь/не успеешь + маршрут на карте
```

## 3. Functional Requirements

### FR-1: Отображение остановок на карте
- **Данные:** Все остановки из GTFS (stops.txt)
- **Визуализация:** Синие точки на карте
- **Interaction:** При наведении — показать имя остановки (tooltip)

### FR-2: Выбор остановки посадки
- **Trigger:** Клик на существующую остановку (синюю точку)
- **Визуализация:**
  - Маркер 🚌 на выбранной остановке
  - Текст "Выберите остановку высадки"
- **Поведение:** Запомнить stop_id, перейти к выбору остановки высадки

### FR-3: Выбор остановки высадки
- **Trigger:** Клик на существующую остановку
- **Визуализация:**
  - Маркер 🏁 на выбранной остановке
  - Текст "Укажите откуда выходите"
- **Поведение:** Запомнить stop_id, перейти к выбору дома

### FR-4: Выбор дома (точка отправления)
- **Trigger:** Клик на произвольную точку карты (после выбора остановок)
- **Визуализация:** Маркер 🏠
- **Поведение:** Запустить поиск автобусов

### FR-5: Поиск автобусов между остановками
- **Trigger:** Выбор stopA, stopB, и homePoint
- **Алгоритм:**
  1. Найти все trips где есть stopA
  2. Для каждого trip: проверить что stopB идёт ПОСЛЕ stopA (stop_sequence)
  3. Найти ближайшее время отправления от stopA (arrival_time > current_time)
  4. Для каждого маршрута: рассчитать wait_time = arrival_time - current_time
  5. Рассчитать walk_time = distance(home, stopA) / 5 км/ч * 60
  6. can_make_it = wait_time > walk_time

### FR-6: UI - Список автобусов
```
✓ 525  через 3 мин (2 мин пешком)
   Метро «Владыкино» → Метро «Отрадное»

✗ 856  через 2 мин (8 мин пешком)
   (не успеешь)
```
- **Элементы:** номер, время, время пешком, галочка/крестик, направление
- **Сортировка:** по времени отправления (возрастание)

### FR-7: Отображение маршрута на карте
- **Визуализация:**
  - Зелёный пунктир: дом → остановка посадки
  - Синий: маршрут автобуса (через все остановки)
- **Автоматически:** Рисуется при выборе дома

### FR-8: Сохранение состояния в URL
- **Параметры:** `?stopA=stop_id&stopB=stop_id&home=lat,lon`
- **Поведение:** При загрузке восстановить точки и показать результат

### FR-9: Сброс
- **Trigger:** Кнопка "Сбросить"
- **Действие:** Очистить всё, вернуть к шагу 1

## 4. Data Structures

### App State
```javascript
{
  stopA: { stop_id, stop_name, lat, lon },
  stopB: { stop_id, stop_name, lat, lon },
  homePoint: { lat, lon }
}
```

### GTFS Files
| File | Fields Used |
|-------|-------------|
| stops.txt | stop_id, stop_name, stop_lat, stop_lon |
| routes.txt | route_id, route_short_name, route_long_name |
| trips.txt | trip_id, route_id |
| stop_times.txt | trip_id, stop_id, arrival_time, stop_sequence |

## 6. Out of Scope (v1)

## 8. Acceptance Criteria

- [ ] Остановки отображаются на карте при загрузке
- [ ] При наведении на остановку показывается её имя
- [ ] Первый клик на остановку → маркер 🚌 (остановка A)
- [ ] Второй клик на остановку → маркер 🏁 (остановка B)
- [ ] Третий клик на карту → маркер 🏠 (дом)
- [ ] После выбора 3 точек показываются автобусы A→B
- [ ] Каждый автобус показывает: номер, время, успеешь/нет
- [ ] Маршрут рисуется на карте (зелёный → синий)
- [ ] URL сохраняет состояние
- [ ] Кнопка сбрасывает приложение
- [ ] Тесты проходят

## 9. Технические заметки

### Формулы
- `walk_time_minutes = distance(home, stopA) / 5 * 60`
- `can_make_it = wait_time_minutes > walk_time_minutes`

### Направление автобуса
- trip содержит stop_times отсортированные по stop_sequence
- Маршрут идёт A→B если: indexOf(A) >= 0 AND indexOf(B) >= 0 AND indexOf(A) < indexOf(B)
